# 스마트 음성인식 가습기

2022년 3월 ~ 2022년 7월 (개인 개발)

* 가습기가 코로나를 포함한 인플루엔자, 바이러스 예방 효과 기사 발견
* 전세계 IoT 관련 사용 기기가 증가하고 있다는 사실에 착안
* 관련 문제 풀이 능력 향상 및 테스트
* 다양한 모바일&PC 환경에서의 IoT 접근 방식 연구 목표

## 활동 내역
- DB 테이블 구조 설계 및 관리
- 웹 서비스 코드 작업 및 서비스 구축
- 아두이노 모듈 연동 회로 설계 및 코드 작업
- 실물 제품 디자인 설계 및 제작
- 버그 테스트 및 FIX 작업
